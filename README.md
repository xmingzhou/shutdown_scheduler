# 多语言定时关机/重启程序

## 功能介绍

这是一个使用Python的tkinter库开发的跨平台应用程序，允许用户设置计算机在指定时间关机或重启。程序支持四种语言：英语、中文、法语和日语，并提供了直观的图形界面，让用户可以轻松设置和管理关机/重启任务。

### 主要功能

1. **多语言支持**：内置英语、中文、法语和日语四种语言，用户可以随时切换界面语言。

2. **定时操作**：
   - 设置精确到分钟的关机或重启时间
   - 支持自定义日期和时间
   - 提供快捷选项（如10分钟后、1小时后等）
   - 一键设置明日早上7:30的关机/重启任务

3. **任务管理**：
   - 显示当前设置的任务状态
   - 随时取消已设置的任务
   - 任务设置成功和取消时的消息提示

4. **跨平台兼容**：支持Windows、Linux和macOS系统，使用系统原生的关机/重启命令。

## 框架结构

程序采用面向对象的设计，主要由以下部分组成：

### 核心类

- `ShutdownScheduler`：主应用类，负责整个程序的初始化和控制
  - 管理UI界面的创建和更新
  - 处理用户输入和事件
  - 执行系统关机/重启命令

### 主要模块

1. **UI界面**：
   - 使用tkinter构建图形界面
   - 包含语言选择下拉菜单
   - 操作类型选择（关机/重启）
   - 日期和时间输入框
   - 快捷操作按钮区域
   - 任务状态显示

2. **多语言支持**：
   - 使用字典结构存储四种语言的界面文本
   - 动态切换语言功能
   - 自动调整按钮大小以适应不同语言的文本长度

3. **系统命令执行**：
   - 根据不同操作系统执行相应的关机/重启命令
   - Windows: 使用`shutdown /s`和`shutdown /r`命令
   - Linux/macOS: 使用`shutdown -h`和`shutdown -r`命令
   - 命令执行错误处理和提示

4. **辅助功能**：
   - 日期和时间解析
   - 任务状态管理
   - 快捷任务计算

## 使用方法

1. 运行程序后，默认显示英文界面
2. 通过顶部的下拉菜单选择所需的语言
3. 选择关机或重启操作
4. 设置日期和时间，或使用快捷按钮
5. 点击"Set Task"设置任务，或"Cancel Task"取消已设置的任务

## 技术细节

- 编程语言：Python 3.9+
- GUI框架：tkinter
- 跨平台支持：Windows、Linux、macOS
- 代码结构：面向对象设计，模块化组织

这个程序充分展示了tkinter在构建跨平台桌面应用方面的能力，同时实现了多语言支持和友好的用户界面。
